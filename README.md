📌 Projeto - Desenvolvimento de Software Multiplataforma (FATEC)

Este repositório contém uma atividade acadêmica desenvolvida como parte do curso de Desenvolvimento de Software Multiplataforma da FATEC.

🧠 Sobre o projeto

O objetivo deste projeto é aplicar, na prática, os conceitos fundamentais de desenvolvimento web, integrando estrutura, estilo e lógica de programação em um ambiente simples, porém funcional.

Aqui não tem firula: é prática de base bem feita — exatamente o que separa quem “brinca de código” de quem constrói de verdade.

🚀 Tecnologias utilizadas

HTML5 → Estrutura da aplicação

CSS3 → Estilização e layout

Node.js → Execução do ambiente backend

🎯 Objetivos da atividade

Compreender a estrutura de uma aplicação web

Praticar a separação entre conteúdo, estilo e lógica

Desenvolver organização de código
